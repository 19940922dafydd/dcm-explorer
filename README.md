# DCM 资源管家 - 桌面版 (Electron)

这是一个已经转换成桌面应用程序（EXE）架构的本地 DICOM 文件检索工具。

## ✨ 桌面版优势

1.  **原生体验**：直接点击 EXE 运行，不再依赖浏览器地址栏。
2.  **更强性能**：文件遍历逻辑在后台进程（Node.js）执行，比浏览器 API 更快、更稳定。
3.  **原生对话框**：使用 Windows 标准的文件夹选择器，更符合操作习惯。
4.  **离线使用**：完全不依赖网络，本地运行。

## 🛠️ 如何生成 .exe 文件

由于我现在为你提供的是源代码，你需要通过以下简单步骤将其打包成 EXE：

### 1. 安装必要的运行环境
确保你的 Windows 电脑上安装了 **Node.js** (下载地址: [nodejs.org](https://nodejs.org/))。

### 2. 初始化项目 (仅需一次)
在项目所在的文件夹内，打开 PowerShell 或 CMD，运行：
```bash
npm install
```
*这会下载 Electron 核心库和打包工具（耗时约 1-3 分钟）。*

### 3. 运行预览 (可选)
如果你想先看看效果，运行：
```bash
npm start
```

### 4. 打包生成 EXE
运行以下命令进行打包：
```bash
npm run build
```
打包完成后，你会发现文件夹中多了一个 `dist` 目录，里面那个 **`DCM资源管家_1.0.0.exe`** 就是你可以直接发给别人或自己双击使用的独立软件了。

## 📂 项目结构说明
- `main.js`: 桌面端的主程序逻辑（窗口管理、文件系统访问）。
- `renderer.js`: 界面交互逻辑。
- `preload.js`: 桥接主程序与界面的安全通道。
- `index.html` & `style.css`: 极致视觉体验的 UI 界面。
- `package.json`: 软件打包配置文件。
